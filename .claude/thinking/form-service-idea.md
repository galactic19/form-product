# Idea: Form 서비스 - 가입상품 영업 전용 신청폼

> 외주 80만원짜리 신청폼을 15만원에. 온라인 영업을 시작하는 가입상품 판매자를 위한 템플릿 + 관리 서비스

---

## The Problem

가입상품(인터넷, TV, 렌탈, 보험 등)을 파는 영업자들은 온라인 채널로 영업을 확장하고 싶지만:

1. **구글폼/네이버폼은 디자인이 구려 신뢰도가 떨어짐** — 고객이 "이거 사기 아니에요?"라고 의심
2. **일반 플랫폼은 가입상품을 지원하지 않음** — 복잡한 조합(인터넷 단독/결합), 요금제 선택, 사은품 조건 등을 처리할 수 없음
3. **외주 개발은 비싸고 수정이 어려움** — 50~100만원 비용 + 상품 변경 시 추가 비용
4. **플랫폼 입점은 수수료 부담** — 가입상품은 결제가 바로 안 되어 플랫폼이 선호하지 않음

---

## The Solution

**가입상품 특화 신청폼 템플릿 + 관리 서비스**

- **업종별 최적화 템플릿**: 인터넷/TV, 렌탈, 보험, 휴폰 등 가입상품에 맞춘 필드와 UI
- **복잡한 조건 처리**: 단독/결합 선택, 요금제 비교, 사은품 조건 등 가입상품 특유의 복잡한 로직 지원
- **전문적인 디자인**: 고객 신뢰를 얻을 수 있는 깔끔하고 신뢰감 있는 폼 디자인
- **플랫폼 독립**: 블로그, 카페, SNS, 광고 등 어디든 링크로 활용 가능

---

## Target Users

**구체적인 프로필:**
- 인터넷/TV/렌탈/보험/정수기 등 가입상품 판매 영업자
- 1인 대리점 ~ 소규모 영업팀 (5인 이하)
- 블로그/카페/SNS를 통해 온라인 영업을 시작하고 싶은 개인 사업자
- IT에 익숙하지 않아 직접 개발은 어려운 40~50대 영업자 포함

**어디서 찾을 수 있는가:**
- 네이버 카페 (인터넷 대리점 카페, 렌탈 영업자 카페)
- 인스타그램/블로그에서 영업하는 개인 판매자
- 네이버 검색광고, 페이스북/인스타그램 광고

---

## Competition & Differentiation

| 경쟁자 | 한계 | 우리의 차별점 |
|--------|------|--------------|
| **구글폼/네이버폼** | 디자인 구림, 가입상품 특화 기능 없음 | 전문 디자인 + 가입상품 특화 로직 |
| **외주 개발사** | 비쌈(50~100만원), 수정 어려움 | 저렴함(15만원), 수정/관리 지속 가능 |
| **해외 폼빌더(Typeform 등)** | 한국어 지원 약함, 카카오톡 연동 없음 | 한국 영업 환경 최적화(카톡 알림 등) |
| **플랫폼 입점** | 수수료 부담, 가입상품 지원 안 함 | 수수료 0원, 독립 브랜드 운영 가능 |

**핵심 차별화 포인트:**
1. **가입상품만의 특수성** 이해 (복잡한 조합/조건 처리)
2. **"만들어 드립니다" 서비스** — 익숙하지 않은 고객도 바로 사용 가능
3. **지속적인 관리** — 템플릿 판매 후에도 수정/업데이트 지원

---

## Riskiest Assumptions

1. **✅ 고객이 구글폼 대신 돈을 내고 템플릿을 살 것인가?**
   - 검증 방법: 광고 테스트 + 초기 고객 피드백
   - 완화책: 가격 조정(10~15만원 범위), 기능 조정

2. **✅ 광고로 지속적인 고객 유입이 가능한가?**
   - 검증 방법: 소규모 광고 테스트로 CAC 측정
   - 완화책: 네이버 블로그/카페 물타입 마케팅 병행

3. **✅ 고객이 "상품 필요성"을 이해할 수 있는가?**
   - 검증 방법: 랜딩페이지 반응률, 문의율
   - 완화책: 메시지 단순화 — "온라인 영업 시작하기"

4. **✅ 수동 제작 비용이 수익성을 해치지 않는가?**
   - 검증 방법: 첫 10건의 실제 제작 시간 측정
   - 완화책: 템플릿 표준화, 반자동화 도구 도입

## MVP Scope

### Phase 1: 템플릿 서비스 (Ver1)

**핵심 기능:**
- [ ] 3~4개 업종 템플릿 (인터넷/TV, 렌탈, 보험, 범용)
- [ ] 업종별 최적화된 폼 필드 구성
- [ ] 모바일 반응형 원페이지
- [ ] 호스팅 + 링크 발급
- [ ] 1회 수정 포함

**수익 모델:**
- 기본 템플릿: 15만원
- 추가 수정: 3~5만원/회
- 구글시트 연동: 10~20만원 (1회)

**성공 기준:**
- 월 20건 판매 (300만원 수익)
- 광고 기준 CAC 확인

### Phase 2: 구독 서비스 (Ver2)

**타이밍:** 템플릿 고객 50명 이상 확보 후

**핵심 기능:**
- [ ] Admin 대시보드
- [ ] 카카오톡 실시간 알림
- [ ] 고객 자동 접수 확인 메시지
- [ ] 폼 무제한 생성/수정
- [ ] 구글시트 자동 연동

**업셀 전략:**
- 템플릿 구매 고객 대상 첫 달 50% 할인

### Phase 3: SaaS 고도화

**장기 비전:**
- [ ] AI 기반 상품 페이지 자동 생성
- [ ] 완전 자동화된 템플릿 제작
- [ ] 스케일 가능한 SaaS 모델로 전환

## Possible Directions

### Direction 1: 고급 에이전시 모델
**설명:** 템플릿을 고급화하여 20~30만원대로 판매. 외주 대비 가성비, 구글폼 대비 퀄리티.

**장점:**
- 단건당 수익 높음
- 고객당 관리 부담 적음 (소수 고급 고객)

**단점:**
- 시장 좁음
- 광고 효율 낮을 수 있음

**선택 시점:** 고급 브랜딩 가능, B2B 영업 강할 때

---

### Direction 2: 볼륨 SaaS 모델 (현재 선택)
**설명:** 15만원 템플릿으로 진입장벽 낮추고, 구독 서비스로 LTV 확장. 광고 기반 지속적 유입.

**장점:**
- 진입장벽 낮음 (고객 확보 용이)
- 구독으로 recurring revenue
- 스케일 가능성

**단점:**
- 초기 수동 작업 많음
- CAC 관리 필요

**선택 시점:** 지금 — 검증된 시장에서 스케일 목표
