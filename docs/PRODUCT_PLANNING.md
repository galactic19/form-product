# Form 서비스 랜딩페이지 기획안 (v2)

---

## 1. 서비스 정의

**한 줄 요약**: 가입상품 판매자를 위한 고객 신청폼 서비스

**상세 설명**: 인터넷, TV, 렌탈, 보험, 정수기 등 "가입"이 필요한 상품을 파는 영업자/대리점이 고객 정보를 받을 수 있는 신청 폼을 저렴하고 빠르게 만들어주는 서비스.

**핵심 키워드**: 가입폼, 신청서, 고객DB, 영업, 가입상품

---

## 2. 타깃 고객 분석

### 누구인가
- 인터넷/TV/렌탈/보험/정수기 등 가입상품 판매 영업자
- 1인 대리점 ~ 소규모 영업팀 (5인 이하)
- 블로그/카페/SNS로 영업하는 개인 사업자
- IT에 익숙하지 않은 40~50대 영업자도 포함

### 현재 어떻게 하고 있나 (As-Is)
- 카카오톡으로 직접 정보를 받음 (누락, 정리 안 됨)
- 네이버 폼이나 구글 폼을 쓰는데 디자인이 구려서 신뢰감이 떨어짐
- 카페/블로그에 댓글로 신청받음 (개인정보 노출 위험)
- 비싼 외주 (50~100만원)로 웹페이지를 만들었다가 수정이 안 됨

### 고객의 핵심 Pain Point (우선순위)
1. **속도** — 영업 기회가 왔는데 폼이 없다. 지금 당장 필요하다
2. **가격** — 외주 맡기면 너무 비싸다
3. **신뢰감** — 구글폼은 고객이 "이거 사기 아니야?" 느낌을 받는다
4. **DB 관리** — 신청 들어오면 바로 알림 + 정리되면 좋겠다
5. **수정 용이** — 상품이 바뀌면 폼도 바로 바꿀 수 있어야 한다

## 3. 상품 구조 (2단계)

### 전체 퍼널

```
[템플릿 구매] ──→ "이거 괜찮네" ──→ [구독 전환]
  5~10만원 1회         신뢰 형성          월 ₩9,900~
```

---

### 상품 A: 원페이지 템플릿 (단건 판매)

**컨셉**: "일단 써보게 하는" 진입 상품

**구성**:
```
┌──────────────────────┐
│   상단 이미지         │  ← 상품 소개 / 혜택 요약 배너
│   (브랜드 신뢰감)     │
├──────────────────────┤
│                      │
│   신청 Form          │  ← 이름, 연락처, 주소, 상품선택 등
│                      │
├──────────────────────┤
│   하단 이미지         │  ← 안내사항 / 주의사항 / 사업자 정보
│   (법적 고지)        │
└──────────────────────┘
```

**가격**:

| 항목 | 가격 | 설명 |
|------|------|------|
| 기본 템플릿 | ₩50,000 ~ ₩100,000 | 업종별 원페이지 (이미지+폼+이미지) |
| 구글시트 연동 | ₩10,000 ~ ₩20,000 (1회) | 폼 응답 → 구글시트 자동 저장 셋업 |
| 이미지 커스텀 | ₩20,000 ~ ₩30,000 (1회) | 로고/색상/문구 맞춤 디자인 |

**포함 내용**:
- 업종별 최적화 원페이지 (모바일 반응형)
- 기본 폼 필드 (이름, 연락처, 주소, 상품 선택)
- 호스팅 포함 (링크 발급)
- 1회 수정 포함

**업종별 템플릿 라인업** (예시):
- 인터넷/TV 가입 신청
- 정수기/공기청정기 렌탈 상담
- 보험 상담 신청
- 휴대폰 개통 신청
- 범용 가입상품 신청

**타깃 고객**: 
- "일단 폼 하나만 있으면 되는" 영업자
- 월 구독은 부담스럽지만, 구글폼보다는 제대로 된 걸 원하는 사람
- IT에 익숙하지 않아서 "만들어주세요" 하고 싶은 사람

**구매 → 구독 전환 시나리오**:
> 템플릿 써보니 신청이 잘 들어옴 → 근데 카톡 알림이 없어서 확인이 늦음 
> → 구글시트 매번 열어봐야 함 → "관리 기능이 있으면 편할텐데" 
> → 구독 상품 전환

---

### 상품 B: 구독 서비스 (월 과금)

**컨셉**: "본격 영업 도구" 프리미엄 상품

**핵심 차별점 (템플릿 대비)**:
1. **Admin 대시보드** — 신청 현황 한눈에 관리
2. **카카오톡 알림** — 신청 즉시 알림 (영업 속도 ↑)
3. **카카오톡 메시지 전송** — 고객에게 접수 확인 자동 발송 (신뢰도 ↑)
4. **폼 무제한 생성/수정** — 상품 바뀌면 바로 수정
5. **구글시트 자동 연동** — 기본 포함

**가격**:

| 플랜 | 월 가격 | 핵심 포인트 |
|------|---------|------------|
| 무료 체험 | 0원 (7일) | 기능 전체 체험, 카드 등록 불필요 |
| 스타터 | ₩9,900/월 | 폼 3개, 월 200건, 카톡 알림, 엑셀 다운 |
| **프로 (추천)** | ₩29,900/월 | 폼 무제한, 응답 무제한, 카톡 메시지, 구글시트, 브랜딩 |
| 팀 | ₩59,900/월 | 프로 + 팀원 5명, 대시보드, API |

**타깃 고객**:
- 매일 영업하는 전업 판매자
- 신청 건수가 많아서 관리가 필요한 사람
- 고객에게 전문적인 인상을 주고 싶은 사람
- 템플릿 써보고 만족해서 업그레이드하려는 사람

---

### 두 상품의 관계

| | 템플릿 (단건) | 구독 (월 과금) |
|---|---|---|
| 성격 | "만들어 드립니다" (납품형) | "직접 관리하세요" (도구형) |
| 가격 | 5~10만원 1회 | 월 ₩9,900~ |
| 진입 장벽 | 낮음 (돈 내고 받으면 끝) | 중간 (가입 후 직접 사용) |
| 수정 | 요청 시 추가 비용 | 직접 자유롭게 수정 |
| 알림 | 없음 (구글시트 확인) | 카카오톡 실시간 알림 |
| 고객 메시지 | 없음 | 카카오톡 자동 접수 확인 |
| 관리 | 구글시트 수동 확인 | Admin 대시보드 |
| 업셀 포인트 | → "알림 있으면 편할텐데" | - |

## 4. 경쟁사 분석

| 서비스 | 월 가격 | 포지션 | 한계 |
|--------|---------|--------|------|
| 구글 폼 | 무료 | 범용 설문 | 디자인 구림, 신뢰감 없음, 알림 없음 |
| Typeform | $25~70/월 | 고급 폼빌더 | 한국어 지원 약함, 영업자에겐 비쌈 |
| Tally | 무료~$29/월 | 노션형 폼 | 한국어 미지원, 가입상품 특화 아님 |
| Jotform | 무료~$39/월 | 범용 폼 | 기능 많지만 복잡함 |
| 왈라 (한국) | 무료~유료 | 한국형 설문 | 설문 특화, 신청서 아님 |
| 외주 제작 | 50~100만원 | 맞춤 제작 | 비쌈, 수정 어려움, 시간 오래 걸림 |

**우리 포지션**: 
- 외주보다 싸고 (템플릿 5~10만원 vs 외주 50~100만원)
- 구글폼보다 전문적이고 (디자인, 신뢰감)
- 해외 서비스보다 한국 영업 현장에 맞춤 (카카오톡, 한국어)

## 5. 랜딩페이지 섹션별 기획

### 유입 경로별 고객 심리

| 경로 | 심리 상태 | 첫 화면에서 봐야 할 것 |
|------|----------|----------------------|
| SNS 광고 | "뭐지?" → 호기심 | 뭘 하는 서비스인지 3초 안에 |
| 블로그/카페 | "이런 게 있었어?" → 탐색 | 내 상황에 맞는지 빠르게 확인 |
| 검색 광고 | "가입폼 서비스" → 비교 | 가격, 기능, 차별점 |

---

### 섹션 1: 히어로

**목표**: 3초 안에 "나한테 필요한 거다" 인식

**헤드라인 후보**:
- A: "가입상품 팔면서 아직도 카톡으로 신청받으세요?"
- B: "고객 신청폼, 5만원이면 완성됩니다"
- C: "영업은 빠른데, 신청서가 느리면 고객을 놓칩니다"

**서브 카피**:
"인터넷, TV, 렌탈, 보험... 가입상품 판매자를 위한 전용 신청폼.
원페이지 템플릿부터 구독형 관리 서비스까지."

**CTA 버튼 2개**:
- 메인: "템플릿 둘러보기" (가벼운 진입)
- 보조: "무료 체험 시작" (구독 바로가기)

**시각 요소**: 
- 모바일에서 원페이지가 보이는 목업
- 상단이미지 → 폼 → 하단이미지 구조가 한눈에 보이게

**심리 패턴**: 
- 질문형 → "맞아, 나 카톡으로 받고 있어" 공감
- CTA 2개 → 진입 경로를 선택하게 해서 이탈 줄임

---

### 섹션 2: 문제 공감

**목표**: "맞아, 나 이거 불편했어"

**섹션 타이틀**: "이런 경험, 있지 않으세요?"

**시나리오 3가지**:

**① 카톡 지옥**
> "이름이요? 연락처요? 주소요? 어떤 상품이요?"
> 하나하나 물어보다 대화 끊기고, 정보 빠지고, 고객 놓침.

**② 구글폼의 배신**
> 링크 보냈더니 "이거 사기 아니에요?" 
> 디자인이 허접해서 고객이 신뢰를 못 함.

**③ 외주의 늪**
> 제대로 만들려니 견적 80만원. 
> 상품 바뀌면 수정비 또 따로. 포기.

**심리 패턴**: 
- 구체적 대화체 → "이 사람 내 상황 아네"
- 3개의 공감 포인트로 하나는 반드시 걸리게

---

### 섹션 3: 해결책 - 두 가지 방법

**목표**: "내 상황에 맞는 게 있네" 선택지 제공

**섹션 타이틀**: "상황에 맞게 선택하세요"

**2-Column 구조**:

```
┌─────────────────────┐  ┌─────────────────────┐
│  💼 템플릿 구매      │  │  🚀 구독 서비스      │
│                     │  │                     │
│  "만들어 드립니다"    │  │  "직접 관리하세요"    │
│                     │  │                     │
│  ✓ 원페이지 완성본    │  │  ✓ 폼 자유롭게 생성   │
│  ✓ 모바일 최적화     │  │  ✓ 카카오톡 알림      │
│  ✓ 링크 바로 사용     │  │  ✓ 고객에게 자동 메시지 │
│  ✓ 1회 수정 포함     │  │  ✓ Admin 대시보드     │
│                     │  │  ✓ 구글시트 자동 연동   │
│  5만원~  (1회)       │  │  월 ₩9,900~          │
│                     │  │                     │
│  [템플릿 보기]       │  │  [무료 체험]          │
└─────────────────────┘  └─────────────────────┘
```

**하단 보충 카피**:
"잘 모르겠다면? → 템플릿부터 시작하세요. 나중에 구독으로 전환할 수 있습니다."

**심리 패턴**: 
- 선택지 제공 → "살까 말까"가 아니라 "어떤 걸 살까"로 전환
- 하단 안내 → 결정 장애 해소, 템플릿으로 유도 (낮은 진입)

---

### 섹션 4: 템플릿 상품 상세

**목표**: "이 가격에 이 퀄리티면 괜찮네"

**섹션 타이틀**: "외주 80만원짜리를 5만원에"

**내용**:

**업종별 템플릿 쇼케이스** (3~4개):
- 🌐 인터넷/TV 가입 신청 템플릿
- 💧 정수기/렌탈 상담 신청 템플릿
- 🛡 보험 상담 신청 템플릿
- 📱 범용 가입상품 템플릿

각 템플릿은 **모바일 목업 이미지**로 보여주기

**가격 안내**:
```
기본 템플릿      ₩50,000 ~ ₩100,000   원페이지 완성 + 호스팅 + 1회 수정
+ 구글시트 연동  ₩10,000 ~ ₩20,000    폼 응답 → 스프레드시트 자동 저장
+ 이미지 커스텀  ₩20,000 ~ ₩30,000    로고/색상/문구 맞춤 제작
```

**보충 카피**:
"주문 후 1~2일 안에 완성. 링크 하나로 바로 영업 시작."

**심리 패턴**: 
- 앵커링 → "외주 80만원" vs "우리 5만원" = 압도적 가성비
- 업종별 쇼케이스 → "내 업종 것도 있네" = 관련성
- 구체적 가격표 → 투명함 = 신뢰

---

### 섹션 5: 구독 서비스 상세

**목표**: "본격적으로 하려면 이게 필요하겠다"

**섹션 타이틀**: "영업이 바쁜 당신을 위한 관리 도구"

**핵심 기능 3가지**:

**① 카카오톡 실시간 알림**
"고객이 폼 작성하면 즉시 카톡으로 알림.
구글시트 열어볼 필요 없이, 바로 전화하세요."

**② 고객에게 자동 접수 확인 메시지**
"신청한 고객에게 카카오톡으로 접수 확인 메시지가 자동 전송됩니다.
'이 업체 제대로 된 곳이네' — 고객 신뢰도가 올라갑니다."

**③ Admin 대시보드**
"모든 신청 내역을 한 화면에서 관리.
처리 상태 체크, 메모 추가, 엑셀 다운로드까지."

**심리 패턴**: 
- "구글시트 열어볼 필요 없이" → 현재 불편함 자극
- 고객 메시지 기능 → "내 고객이 나를 더 신뢰하게 된다" = 소유욕

---

### 섹션 6: 가격표 (통합)

**목표**: 전체 상품을 한눈에 비교, 즉시 결정

**섹션 타이틀**: "커피 한 잔 가격으로 시작하세요"

**구성**: 탭 또는 2영역으로 나눠서 보여주기

**영역 A: 템플릿 (1회 구매)**

| 항목 | 가격 |
|------|------|
| 기본 템플릿 | ₩50,000 ~ ₩100,000 |
| + 구글시트 연동 | ₩10,000 ~ ₩20,000 |
| + 이미지 커스텀 | ₩20,000 ~ ₩30,000 |

"빠르게 하나만 필요한 분에게 추천"

**영역 B: 구독 서비스 (월 과금)**

### 섹션 7: 이렇게 작동합니다

**목표**: "진짜 쉽겠네" 행동 장벽 제거

**섹션 타이틀**: "시작하는 건 간단합니다"

**템플릿 구매 플로우**:
```
STEP 1              STEP 2              STEP 3
업종 선택        →   정보 전달        →   링크 수령
"파는 상품을          "로고, 연락처,       "1~2일 안에
 알려주세요"           원하는 문구"         완성본 전달"
```

**구독 서비스 플로우**:
```
STEP 1              STEP 2              STEP 3
가입              →   폼 만들기       →   공유
"30초면 끝"          "템플릿 선택하고       "링크 복사해서
                      수정하면 완성"        블로그/SNS에 붙이기"
```

**심리 패턴**: 
- 3단계 = "이것만 하면 된다" 단순화
- 구체적 시간 ("30초", "1~2일") → 예측 가능 = 안심

### 섹션 11: CTA (최종 행동 유도)

**목표**: 지금 당장 행동

**헤드라인**: "다음 고객을 놓치기 전에"

**서브 카피**: "템플릿 하나면 내일부터 영업이 달라집니다"

**CTA 2개**:
- 메인: "템플릿 주문하기 →" (5만원~)
- 보조: "구독 무료 체험 →" (7일)

**하단 신뢰 요소**: 
"1~2일 내 완성 · 카드 등록 불필요 · 100% 환불 보장"

**심리 패턴**: 
- "다음 고객을 놓치기 전에" → 손실 회피
- CTA 2개 → 어느 쪽이든 행동하게 만들기
- "내일부터" → 즉시 결과 기대감

## 6. 전체 메시지 전략

### 핵심 메시지

**템플릿**: "외주 80만원짜리 신청폼을 5만원에. 내일부터 바로 영업."
**구독**: "신청 들어오면 카톡으로 바로 알림. 영업 기회를 놓치지 마세요."

### 톤 & 보이스
- **직설적** — 영업자는 바쁘다. 돌려 말하지 않는다
- **현실적** — 과장 없이 실제 상황 기반
- **동료 느낌** — "저도 영업 해봐서 알아요"

### 절대 안 쓸 단어
- "혁신적인 AI 기반 솔루션"
- "디지털 트랜스포메이션"
- SaaS, CRM, API 등 전문 용어 (고객 대상 페이지에서)

### 반복 키워드
- 가입상품, 신청폼, 카카오톡 알림, 고객 신뢰
- 5만원, 3분, 내일부터, 바로

## 7. 심리 패턴 적용 맵 (최종)

| 섹션 | 구매 심리 | 적용 패턴 | 효과 |
|------|----------|----------|------|
| 히어로 | 문제 인식 | 질문형 + CTA 2개 | "나한테 필요한 거다" |
| 문제 공감 | 불안 | 구체적 대화 시나리오 | "이러면 안 되겠다" |
| 선택지 | 결정 유도 | 2가지 경로 제시 | "어떤 걸 살까" (이탈 방지) |
| 템플릿 상세 | 가성비 확신 | 앵커링 (80만원 vs 5만원) | "이 가격이면 당장" |
| 구독 상세 | 소유욕 | "카톡 알림" = 놓치지 않는다 | "이거면 영업이 편해지겠다" |
| 가격표 | 즉시 결정 | 프로 플랜 강조 + 전환 할인 | "프로가 가성비 좋네" |
| 작동 방식 | 자기효능감 | 3단계 플로우 | "나도 할 수 있겠다" |
| 미리보기 | 욕구 강화 | Before/After | "내 것이면 좋겠다" |
| 후기 | 사회적 증거 | 업종별 + 전환 스토리 | "나 같은 사람이 쓰네" |
| FAQ | 의심 해소 | 리스크 제거 문답 | "걱정할 거 없네" |
| CTA | 행동 유도 | 손실 회피 + CTA 2개 | "안 하면 손해" |
